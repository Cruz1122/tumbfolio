# Documentación técnica Tumbfolio

Esta carpeta define la base arquitectónica del entregable `T-01`.

## Índice

- `architecture.md`: arquitectura base y fronteras entre frontend, backend, workers y dominio.
- `project-structure.md`: estructura de carpetas y responsabilidades.
- `environment.md`: variables de entorno y servicios locales.
- `makefile.md`: comandos normalizados del proyecto.
- `stack.md`: versiones fijadas del stack.
- `security.md`: decisiones negativas de seguridad desde el scaffold.
- `t-01-acceptance.md`: criterios de aceptación del entregable técnico 1.
- `adr/`: decisiones arquitectónicas registradas.
- `reference/`: documentos fuente usados como referencia de producto y roadmap.
