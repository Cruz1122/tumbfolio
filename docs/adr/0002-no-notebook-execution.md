# ADR 0002 — No ejecutar notebooks en el MVP

## Decisión

El MVP leerá notebooks ya ejecutados. No abrirá kernels, no ejecutará celdas y no instalará dependencias del usuario.

## Consecuencia

Se reducen riesgos de seguridad, costos de cómputo y complejidad de ambientes. El valor del producto se concentra en curar visualmente resultados existentes.
